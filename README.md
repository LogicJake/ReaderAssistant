# ReaderAssistant
## 解决的问题
打开pdf时自动跳到上次阅读的位置，方便管理自己的pdf阅读
## 特点  
* 管理本地的pdf文件
* 应用内阅读pdf文件
* 阅读时支持跳转
* 退出阅读自动记住页码，下次打开直接跳转到该页
* 阅读记录可以本地备份
* 换设备后可以通过导入备份文件恢复阅读记录
